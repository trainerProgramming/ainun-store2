# AinunStore

AinunStore adalah website **e-commerce digital sederhana** yang dibuat menggunakan **HTML, CSS, dan JavaScript**. Aplikasi ini memungkinkan pengguna untuk melihat daftar produk digital, menambahkan produk ke keranjang, serta mengelola item yang akan dibeli melalui antarmuka yang responsif dan mudah digunakan.

## 🚀 Fitur Utama

### 📦 Daftar Produk

- Menampilkan katalog produk digital.
- Informasi produk berupa:
  - Nama produk
  - Deskripsi singkat
  - Harga
  - Gambar produk

### 🛒 Keranjang Belanja

- Menambahkan produk ke keranjang.
- Menghapus produk dari keranjang.
- Mengubah jumlah produk.
- Menghitung total harga secara otomatis.

### 📱 Responsive Design

- Tampilan menyesuaikan berbagai ukuran layar.
- Optimal untuk desktop, tablet, dan mobile.

### ⚡ Interaktif dengan JavaScript

- Pengelolaan data keranjang secara dinamis.
- Perhitungan total harga otomatis.
- Update tampilan tanpa perlu refresh halaman.

## 📖 Cara Menjalankan Project

1. Download atau clone repository.

```bash
git clone https://github.com/username/ainunstore.git
```

2. Buka folder project.

```bash
cd ainunstore
```

3. Jalankan file:

```text
index.html
```

menggunakan browser seperti:

- Google Chrome
- Mozilla Firefox
- Microsoft Edge

## 🎯 Tujuan Project

Project ini dibuat sebagai latihan pengembangan website e-commerce menggunakan teknologi frontend dasar tanpa framework, sehingga cocok untuk mempelajari:
